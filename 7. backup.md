### **အဆင့် 2: Backup & Verification Process သတ်မှတ်ပါ**

| လုပ်ဆောင်ချက် | တာဝန်ရှိသူ | နည်းလမ်း |

| **Verification လုပ်ခြင်း** | သင် (သို့) IT Assistant | - Backup File ရှိမရှိ စစ်ဆေးခြင်း<br>- File Size မှန်မမှန်<br>- Test Restore (လစဉ် တစ်ကြိမ်) |

> ⚠️ **“Run လုပ်တာ” မဟုတ် — “Verify လုပ်တာ” ဖြစ်ရမည်**  
> → ဥပမာ: “Backup လုပ်ပြီးပြီ” မဟုတ်ဘဲ “Backup file ကို ဖွင့်ကြည့်ပြီး ဒေတာပါဝင်မှု အတည်ပြုပြီး”

## 💡 **အောင်မြင်မှုအတွက် အကြံပြုချက်များ**

1. **Auto-Backup စနစ် သုံးပါ** – ဥပမာ: Cron Job (Linux), Task Scheduler (Windows)
